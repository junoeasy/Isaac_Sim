## 1. 로보틱스를 위한 선형대수

### 공부할 내용
- 벡터
- 행렬
- 내적
- 외적
- 역행렬
- 좌표축
- 동차좌표

### 아래 표현에 익숙해져야함
``` python
p_world = T_world_robot @ p_robot
```
의미 : robot frame에서 표현된 점 `p_robot`을 world frame 기준의 점 `p_world` 로 변환.
이때 사용하는 변환 행렬이  `T_world_robot`이다.

### 벡터
- 방향과 크기를 가진 값
`v = [1,0]`
	- 2D에서 x방향으로 1만큼 가는 벡터
`v = [0,1]`
	- 2D에서 y방향으로 1만큼 가는 벡터
### 행렬
숫자를 직사각형 형태로 배열한 것
```
A=
[1 2]
[3 4]
```
로보틱스에서는 행렬을 회전 / 이동 / 좌표계 변환 / 카메라 projection / 로봇 Jacobian / 신경망 계산 등에 사용
2D 회전행렬 형식
```
R=
[ cos(theta) -sin(theta) ]
[ sin(theta)  cos(theta) ]
```

### 내적
내적은 두 벡터가 `얼마나 같은 방향을 보고 있는지` 측정하는 계산
공식
```
a · b = a_x * b_x + a_y * b_y
a · b = |a||b|cos(theta)
```
ex1)
```
a = [1,0]
b = [1,0]
a · b = a_x * b_x + a_y * b_y 
= 1*1 + 0*0 
= 1 + 0 = 1
# 둘이 같은 방향을 보고 있다.
```
ex2)
```
a = [1,0]
b = [0,1]
a · b = a_x * b_x + a_y * b_y 
= 1*0 + 0*1 
= 0 + 0 = 0
# 둘이 수직이다.
```
ex3)
```
a = [1,0]
b = [-1,0]
a · b = a_x * b_x + a_y * b_y 
= 1*(-1) + 0*0 
= -1 + 0 =-1
# 서로 반대 방향이다.
```
사용환경
- 로봇 손이 목표 방향을 잘 향하고 있는가?
- 카메라가 물체를 바라보고 있는가?
- 두 방향 벡터 사이 각도가 얼마인가?
### 외적
두 벡터에 수직인 벡터를 만드는 연산. 주로 3D에 쓰임
공식
```
|a x b| = |a||b|sin(theta)
```
ex)
```
x축 = [1, 0, 0]
y축 = [0, 1, 0]

#외적
x축 x y축 = z축

#결과
[0,0,1]
```
사용환경
- 회전축 계산
- 법선 벡터(Normal vector) 계산
- 토크 계산
- 좌표계의 세번째 축 만들기

### 역행렬
행렬의 되돌리기이다.
숫자에서 1에 3을 곱해서 3을 만들었을 떄 다시 돌리려면 3의 역수인 1/3을 곱하면 되는 것처럼
행렬에서도 행렬에 역행렬을 곱하면 단위행렬이 된다.
```
A @ A_inv = I
# I는 단위행렬
# @는 행렬 곱하기
```

### 좌표축
위치와 방향을 설명하기 위한 기준
좌표축은 하나만 있는게 아니라 여러 개가 동시에 존재
ex)
```
world frame
robot base frame
end-effector frame
camera frame
object frame
```
### 동치좌표
좌표 뒤에 `1`을 붙인 표현
ex)
```
# 2D 점
p = [x,y]

#동치좌표
p = [x,y,1]

# 3D 점
p = [x,y,z]

#동치좌표
p = [x,y,z,1]
```
1을 붙이는 이유
- 회전과 이동을 행렬 하나로 같이 처리하기 위해서
- 3D 공간에서 어떤 점을 이동시키고 회전시키는 연산이 많이 발생하는데 회전은 행렬 곲(3X3), 이동은 벡터 더하기(3X1)로 연산 포맷이 다른데, 동차좌표를 써서 (4X1)로 늘리면 4x4로 합칠수 있게 되어 계산이 편해짐
