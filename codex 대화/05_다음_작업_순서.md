## 2. joint2 회전축 확인

Stage에서:

```text
Joints
  joint2_shoulder_test
```

를 선택한다.

오른쪽 Property에서:

```text
Revolute Joint > Axis
Drive > Angular > Target Position
```

을 확인한다.

테스트:

```text
Target Position = 20 또는 30
Play
```

축이 이상하면:

```text
Axis X -> Y -> Z
```

순서로 바꿔본다.

## 3. joint2가 맞으면 joint3 테스트 파일 만들기

joint3 테스트는 다음 구조가 되어야 한다.

```text
base_link 고정
link1 joint1으로 연결
link2 joint2로 연결
link3 joint3로 연결
link4~link6 숨김
```

joint1, joint2는 0도 또는 원하는 고정 자세로 둔다.

joint3만 Target Position을 바꿔 테스트한다.

## 4. 같은 방식으로 joint4~joint6 진행

한 번에 6개를 전부 켜지 말고, 하나씩 늘린다.

```text
joint1 성공
joint2 성공
joint3 성공
joint4 성공
joint5 성공
joint6 성공
```

## 5. 전체 articulation 파일 만들기

모든 joint가 따로 성공하면 전체 파일을 만든다.

전체 파일에는 다음이 필요하다.

```text
ArticulationRoot
PhysicsScene
base_link~link6 RigidBody
각 link collision
world_fixed_base Fixed Joint
joint1~joint6 RevoluteJoint
각 joint Angular Drive
mass/inertia 값
```

## 6. 최종 튜닝

서보처럼 만들기 위해 조정할 값:

```text
Stiffness
Damping
Max Force
Joint limit
Velocity limit
Mass
Inertia
Collider approximation
```
